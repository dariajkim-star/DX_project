## 1. P-2(설정 휘발) 정규식 정밀화 — 오탐 16.1% 적발

**경위**: Grumbal(파티)이 "P-2 매칭 448건, 눈으로 검수한 사람 있어?"라고 문제 제기.
샘플 20건 검수 결과 긍정 리뷰 혼입 확인.

- 초안 정규식은 "제품 등록"의 단순 언급까지 매칭 → 매칭 484건 중 **별점 4~5점이 78건(16.1%)**
  (예: "제품등록이 쉽네요", "여러 제품을 등록하고 사용 시험 중입니다")
- **소실·재작업을 명시한 표현만** 남기도록 패턴 축소
  (`재등록`, `첨부터 다시`, `등록…날아/사라/삭제`, `설정…초기화` 등 — `compare_competitor.py` 참조)
- 결과: ThinQ 2.62%(448건) → **1.37%(235건)**, 오탐률 16.1% → 11.3%. 경쟁 배수 2.9배 → **2.0배**
- 재검수 15건 샘플은 전량 진성 P-2 확인 ("업데이트했더니 등록했던거 삭제되고 다시 등록도 안됨",
  "강제업데이트하면 기존연결 다 풀리고", "공유기 바꿨는데 새 네트워크에 기기등록이 안돼서")

**판정**: P-2는 소수 노이즈가 아닌 실재 Pain이나 P-1에 비하면 부차적. 헤드라인 1순위는 P-1,
P-2는 "경쟁 열위가 명확한 차별화 근거"로 배치.

## 2. 페르소나 P1 가상 인물 설정 삭제 — 인구통계 부재 확인

**경위**: 다리아가 "응답자 연령·성별 데이터를 나한테 알려준 적 있냐"고 추궁 → Mary 자수.

  "인구통계 미상 — 야간에 아기방을 지키는 LG 풀가전 가구"로 축소
