## 1. 결정 사항

### 1.1 네이버 데이터는 토픽 분석에서 제외한다
**근거 (Sally·Mary)**
- 텍스트의 목적이 리뷰와 반대다. 리뷰 = 감정 배출("밤마다 먹통임"), 네이버 = 문제 해결 가이드
  ("연결 오류 2.4GHz 설정으로 3분 만에 해결"). 같은 토픽 모델에 넣으면 "연결"이 불만인지
  해법인지 구분되지 않아 언급률이 오염된다.
- 협찬 블로그에 감성분석을 돌리면 긍정이 나온다 — VOC가 아니다.
- 대신 네이버는 **여정(journey) 서사와 정성 증언의 광맥**이다. 리뷰엔 없는
  "그날 밤 무엇을 시도했고 무엇이 실패했는가"가 기록돼 있다.

**용도 확정**: P-2 심층 분석 + 정성 인용 추출 전용. BERTopic·감성분석 대상에서 제외.
**Grumbal 조건**: "1만 건을 왜 버렸냐"는 질문에 답할 근거를 문서에 남길 것 → 이 절이 그 근거.

1. ThinQ 리뷰 17,446건 단독 재분석 (파라미터 스윕 포함) — † 표기 수치 갱신이 최우선
2. 네이버는 정성 추출 파이프라인 별도
3. SmartThings 토픽 분석은 3순위(선택) — 경쟁 대조는 이미 1점 비율·P-2 배수로 충분

## 2. 실행 결과 — v2 수집 (2026-07-21)

**32,183건** 수집 (중복 8,832·스마트폰 오염 578 제거 후). 소스: 블로그 14,878 / 카페 10,514 / 지식iN 6,791.

### 2.2 유효율 측정 기준의 정정 (중요)
초안 측정은 "축 시그널 + 브랜드어(씽큐|ThinQ|**LG**|엘지)"로 판정해 **40~60% 전 축 채택**이 나왔다.
그러나 눈검수 결과 대량의 노이즈가 확인됨:
- 인터넷 가입 광고 ("양산시 강서동 LG SK KT 인터넷 가입 현금 사은품")
- LG전자 주식 분석 ("5년 중 최저 PBR 23년 연속 배당")
- 무관 콘텐츠 ("씽큐베이션" = 독서모임 서평)

→ 판정 기준을 **앱 자체 언급(씽큐|ThinQ)** + **오프토픽 제외**로 강화. 유효율이 절반 이하로 떨어짐.
이는 P-2 정규식 정밀화(오탐 16.1% 적발)와 같은 유형의 교훈이며, **자동 지표는 반드시
눈검수로 교차 확인한다**는 원칙을 재확인한 사례다.

### 2.3 축별 해석
- **C_경쟁대안(20.9%, 채택)**: 유일하게 기준 통과. "LG씽큐, 구글 홈, 애플 홈킷 등과 연동",
  "스마트싱스의 진짜 매력은…" 같은 **생태계 비교 담론**이 실재한다 → 처방(개방 생태계·온바디)의
  시장 근거로 사용 가능.
- **A·B·D(8.9~11.1%)**: 키워드가 앱이 아니라 **가전 제품·설치 서비스** 담론을 끌어옴.
  "이사"는 이삿짐/이전설치 서비스 글로, "가족 공유"는 인터넷 결합 상품 글로 흘렀다.
  → 재설계 방향: 앱명을 쿼리에 **강제 포함**(예: `씽큐 이사 재등록`처럼 제품어를 앱명으로 고정),
  또는 지식iN 소스 비중 확대(질문글이 앱 사용 맥락에 더 밀착).
