"(내부 데이터 분석, 2023)"을 붙임 → 재실행 1회 허용(조건: 재불량 시 산출 불가 확정)

신뢰 불가로 확정되어, 수치 채택 대신 **"종합 신뢰도: 산출 불가(신뢰도 근거 불량)"**

기각된 것은 B의 자기 신뢰도 평가뿐이다.

> 발표 활용(파티 결정): 이 적발 과정 자체가 "AI는 초안, 책임은 사람" 원칙과

## 4. 문제 정의 (POV + HMW)

**POV** (1위 토픽 + CX_DEFINITION 실증 수치 기반):
> LG 가전을 여러 대 보유한 사용자는 **전원·네트워크·서버 상태가 바뀔 때마다 홈
> 프로필이 무력화되는 경험**(부정 VOC의 43.4%)을 하며, 이는 개별 기기 버그가 아니라
> **홈 상태가 클라우드·앱에 종속된 구조** 때문이다.

**HMW 3개:**
1. HMW-1. 서버·앱이 죽어도 **집의 상태가 유지되는** 홈 프로필을 어떻게 만들까?
   (P-1, 43.4% — 온바디 프로필 + BLE 로컬 제어 직결)
2. HMW-2. 이사·재설치·전원 차단 후 **재등록 없이 프로필이 따라오게** 하려면?
   (P-2 설정 휘발 1.37%, 경쟁 2.0배 — "Home Follows You" 핵심)
3. HMW-3. 개인정보를 **덜 내고도** 개인화된 홈을 어떻게 제공할까?
   (P-3 — **주 수치 6.2%**: 정밀 패턴 매칭+눈검수, 파티 결정 2.
   각주: LDA 토픽 배정 기준으로는 0.5% — 토픽 모델은 보수적으로 배정하며,
   두 수치의 기준 차이는 발표 자료에 병기)

조사 항목으로 강등했다"* — 미검증을 미검증으로 다루는 시스템의 증거.
